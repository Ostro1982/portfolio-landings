# Portfolio · Landing Pages

Three landing page demos for fictional clients, each built with a deliberately different structural and visual identity. Designed as a portfolio piece for freelance platforms.

**Live demo:** https://ostro1982.github.io/portfolio-landings

---

## What's inside

### 00 · Portfolio Index — `/`
**Layout:** Terminal / CLI window interface
**Why:** Positions the developer as a builder, not a template-flipper. Sidebar file tree, monospace everywhere, command prompts as section headers.
**Stack:** HTML + CSS only. JetBrains Mono.

### 01 · Lumen Aesthetic Medicine — `/clinica`
**Layout:** Centered single-column longread (max 660px). No grids, no sidebars.
**Why:** A high-end aesthetic clinic should feel like reading a luxury print magazine, not browsing a service menu. Treatments are listed sequentially. Pull quotes break the column.
**Stack:** HTML + CSS only. Cormorant Garamond + Inter Tight.

### 02 · The Brasa & Salt Daily — `/parrilla`
**Layout:** Newspaper broadsheet — masthead, 4-column grid, varying article sizes, drop caps.
**Why:** A steakhouse with strong opinions should publish them like a newspaper. Front-page lead, info sidebar, menu as classifieds, pull quotes, end rule.
**Stack:** HTML + CSS only. Playfair Display + Lora + Old Standard TT.

### 03 · MATA Studio — `/productora`
**Layout:** Cinema viewer interface — fixed top REC bar, bottom timeline scrubber with chapter markers, letterboxed sections, chapter cards.
**Why:** A film studio's site should feel like film equipment. Title screen, chapter markers, end credits, frame grid for the reel.
**Stack:** HTML + CSS only. Archivo + Inconsolata.

### 04 · Cadence — `/cadence`
**Layout:** Editorial SaaS dashboard — magazine-style asymmetric grid for an AI LinkedIn ghostwriter app. Hero headline, featured post with drop cap and projected stats, voice profile sidebar with tone calibration, six-day card grid, generation engine card with offset shadow.
**Why:** A writing tool deserves a writing-tool interface — typography-driven, paper-textured, opinionated. Built as both a portfolio piece and a viable product mockup.
**Stack:** HTML + CSS only. Instrument Serif + Newsreader + Geist + Geist Mono.

---

## Stack

- HTML5 + CSS3 only — no frameworks, no build step
- Google Fonts via CDN
- Mobile-first responsive
- Animations: pure CSS
- Zero JavaScript dependencies
- Deploy: GitHub Pages

## Philosophy

Each demo has a **completely different layout and structure**, not just different colors. The same developer can think in different visual languages depending on the client's business — that's the point of this portfolio.

No component libraries. No Bootstrap. No Tailwind. Just CSS designed for each case.
